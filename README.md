# Cerveau — прототип

[![Deploy to GitHub Pages](https://github.com/dzhokhar1/sabrina/actions/workflows/deploy.yml/badge.svg)](https://github.com/dzhokhar1/sabrina/actions/workflows/deploy.yml)

🔗 **Живой сайт:** https://dzhokhar1.github.io/sabrina/

Демонстрационный макет платформы персонализированного обучения. Человек
проходит «тест» на тип восприятия (Визуал / Аудиал / Кинестетик) и получает
лекцию в подходящем формате: инфографика, аудио или интерактивные задания.

> Прототип. Все «умные» части (ИИ, психологический портрет, уровень знаний) —
> заранее подготовленные заглушки. Бэкенда нет, данные пользователя хранятся в
> `localStorage`. Язык интерфейса — русский.

## Стек

- **Vite + React 19 + TypeScript** — SPA, статическая сборка
- **Tailwind CSS v4 + shadcn/ui** (preset Nova / Radix) — UI-компоненты
- **React Router** — маршрутизация
- **react-hook-form + zod** — форма регистрации
- **framer-motion** — анимации (в т.ч. «ИИ думает…»)
- **@fontsource-variable** — Playfair Display (заголовки) + Inter (текст), с кириллицей

## Дизайн-система

Редакционный стиль («Editorial Intelligence»): serif-дисплей на тёплом бумажном
фоне, чернильно-синий как доминанта, три акцента под типы восприятия (индиго /
лазурь / фиолет). Все токены — в [`src/index.css`](src/index.css).

## Запуск

```bash
npm install
npm run dev        # http://localhost:5173
npm run build      # сборка в dist/ (статика)
npm run preview    # предпросмотр сборки
```

## Структура

- `src/pages/` — экраны (регистрация, кабинет, портрет, уровень, библиотека, курс, лекция, загрузка, сотрудничество)
- `src/components/` — layout, общие компоненты, `ai/AiThinking`, `result/*` (три формата), `media/AudioPlayer`, `upload/Dropzone`
- `src/data/` — «зашитые» данные: курсы, портрет, уровень, задания, метаданные типов восприятия
- `src/context/UserContext.tsx` — пользователь в `localStorage`
- `public/demo/lecture-audio.m4a` — демо-озвучка (русский TTS) для аудиал-формата

## Демо-сценарий

1. На главной заполните анкету, выберите тип восприятия → «Войти».
2. В кабинете откройте курс → «Получить в моём формате».
3. После анимации «ИИ думает…» материал появится в вашем формате.
4. Попробуйте «Загрузить свою лекцию» — результат тот же по типу восприятия.

Чтобы сменить тип восприятия — выйдите (иконка в шапке) и зарегистрируйтесь заново.

## Что осознанно отложено

Настоящий ИИ-движок, реальные тесты (тип восприятия, портрет, уровень), оплаты,
админка, мобильное приложение, английский язык, вход по паролю и email.
